Introduction to the Web
The World Wide Web also known as the Internet is made up of over 30 Billion Documents. Although you can put any type of file on the internet the most widely found type is HTML. HTML means Hyper Text Markup Language.

File Types

The Componenets of the Web
The major pieces of the Web consist of You (this includes your computer and the browser), the internet, http, and the servers.
browser - is a program that runs on your computer to display files found on the Web.
internet - is the worlds biggest network.
http - the main protocol of the Web.
Servers - computers that host the files that make up the Web.

Intro to HTML
HTML is made up of:
Text Content - What you see
Markup - How it looks
References to files - i.e. videos, images, and other documents
Hyperlinks or Links - these connect you to other pages

HTML Tags
HTML Tags determine how your page looks and look like this < opening tag>content< /Closing tag>. Different tags do different things such as BOLD or italicize. The whole tag is called an element.
